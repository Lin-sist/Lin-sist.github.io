大二下2月规划

发表于2026-02-07|更新于2026-02-12|随笔感悟
|浏览量:

引入“费曼测试”作为强制验收

觉得“完成得很水”，那我们就加一道“防伪大坝”。 每天的【核心任务】完成后，必须做一个动作：向AI提问或复述。

  • 指令：“Copilot/Gemini，我刚写完了登录模块，我现在用一句话总结它的逻辑：xxxx。请你检查我的理解有没有漏洞，并充当面试官追问我一个相关坑点。”

  • 验收标准：只有当你能回答上 AI 追问的那个坑点，今天的任务才算画上句号。

1. 二月核心目标（必须完成，其他的都是赠品）

  • 项目闭环：彻底吃透《苍穹外卖》核心业务，并完成“AI 菜品智能生成”或“智能客服”的功能魔改（你的简历亮点）。

  • 技术攻坚：掌握 Redis 缓存（穿透/击穿/雪崩）和分布式锁（基于《黑马点评》逻辑）。

  • 算法保底：Hot100 刷完前 20 题（数组、链表、哈希），能白板默写。

  • 简历初稿：3月1日必须写出第一版简历。

2. 详细任务清单（2月8日 - 3月2日）

阶段一：启动与适应期（2月8日 - 2月11日）
策略： 每天只做最重要的 1 件事，适应 Ubuntu + AI 工作流，找回手感。

2026-02-08（周日）

  • 【保底】 打开 Ubuntu，在 IDEA 里运行一次《苍穹外卖》，确保无报错。

  • 【额外】配置obsidian,配置fictx5输入法。

  • 【核心】 项目复盘：利用 Copilot Plan 模式，梳理“员工管理”和“分类管理”的业务流程。尝试删掉 EmployeeServiceImpl 的 save 方法，自己重写一遍。

  • 【算法】 Hot100 做 1 题（160.相交链表）。Gemini 引导 -> 伪代码 -> 默写。

2026-02-09（周一）

  • 【保底】 读一遍昨天的算法题代码。

  • 【额外】整理obsidian笔记。

  • 【核心】 项目复盘：公共字段自动填充（AOP）。问 Copilot：“面试官问我 AOP 在项目里怎么用的，我该怎么回答？”

  • 【算法】 Hot100 第 206 题（反转链表）。

2026-02-10（周二）

  • 【保底】 打开 VS Code，看一眼 Redis 的连接配置。

  • 【额外】 配置claude code。

  • 【核心】 项目复盘：店铺营业状态设置（Redis 入门）。这是你第一次在项目里正儿八经用 Redis，搞懂 RedisTemplate 的用法。

  • 【算法】 Hot100 第 234 题（回文链表）。

2026-02-11（周三）

  • 【核心】 项目攻坚（重难点）了解 JWT 登陆认证模块核心业务逻辑。利用“费曼技巧”给 Copilot 复盘。

  • 【保底】 通读copilot解答+阅读相关类完整源码。

  • 【算法】 复习前 3 天的 3 道题（白板默写）。

阶段二：高强度攻坚期（2月12日 - 2月15日）
策略： 这 4 天是寒假效率最高的时候，我们要拿下 Redis 和 MySQL 优化。

2026-02-12（周四）

  • 【核心】 引入《黑马点评》知识点：学习“缓存击穿/穿透/雪崩”理论。不需要写点评的代码，直接在《苍穹外卖》的“查询菜品”接口里实战这些逻辑。

  • 【算法】 Hot100 第 49 题（字母异位词分组）。

2026-02-13（周五）

  • 【核心】 项目魔改（亮点启动）：注册 DeepSeek 或 OpenAI API。在项目里写一个简单的 Test 类，调通 API，让它能回答“帮我推荐一个川菜”。

  • 【算法】 Hot100 第 128 题（最长连续序列）。

2026-02-14（周六）

  • 【核心】 项目魔改：设计“AI 菜品描述生成”功能。在添加菜品时，让 AI 自动生成诱人的描述文案。

  • 【算法】 Hot100 第 283 题（移动零）。

2026-02-15（周日）

  • 【周复盘】 检查本周算法题是否都能默写。

  • 【休息】 战术性挂机。只做保底：看一篇 Java 基础八股文（HashMap）。

阶段三：春节低功耗模式（2月16日 - 2月20日）
策略： 预设这几天很忙。只做保底，绝不强求。 只要不断链就是胜利。

2026-02-16（周一·除夕前） ~ 2026-02-20（周五·初三）

  • 【每日保底】 手机上下个“力扣 App”或者“CSDN App”。每天蹲坑时看 1 个面试题（如：TCP 三次握手、MySQL 索引结构）。

  • 【每日保底】 睡前在脑子里过一遍《苍穹外卖》的登录流程。

(这几天不需要打开电脑，好好过节/休息，不要有负罪感)

阶段四：最后冲刺与简历（2月21日 - 3月2日）
策略： 收心，把项目打包，写简历。

2026-02-21（周六）

  • 【保底】 打开 Ubuntu，更新代码。

  • 【核心】 恢复训练：复习 Redis 缓存逻辑。

  • 【算法】 Hot100 第 15 题（三数之和）。

2026-02-22（周日）

  • 【核心】 项目收尾：确保你的“魔改功能”（AI生成/智能客服）能跑通。录一个 30 秒的 Demo 视频（防备面试演示）。

  • 【简历】 找一个 Markdown 简历模板，把《苍穹外卖》写上去。重点写：使用 Redis 解决了xx问题，接入 LLM 实现了xx功能。

2026-02-23 ~ 2026-03-01（开学前一周）

  • 【每日任务】 每天 2 道 Hot100（链表专题）。

  • 【每日任务】 每天背 3 道 Java 基础八股文。

  • 【核心】 模拟面试：把简历发给 Gemini，让它扮演面试官，针对你的项目提问。
